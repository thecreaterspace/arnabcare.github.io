Grape OS

Open Source, Defined

Grape is a collection of Operating Systems that are open source and developer-made.
They have an approchable UI but can be accesed to the core.

Fully customizeable, the way you want it.

Grape is made in many forms. It can be used as Linux, a custom OS, or a tester.
This makes it very usefull because it can be used on many systems.
